Portfolio

CSS Web App

This app was created using Angular JS, Node.JS, jQuery, CSS3, HTML5, and MongoDB. The user is able to generate the CSS for two color gradients. The colors can be changed by clicking the small boxes, which opens up a color picker. The boxes can also move to adjust the position of the gradient. The direction can be modified by clicking on the coordinating arrows. If the user is logged in, they have the option to save the gradient to their profile, which they can then access later.

Wuollet's Bakery

This website was created using PHP, MySQL, CSS3, jQuery, and HTML5. The website is a class project that I completed by redesigning a bakery in Minnesota's website. I continued this project and created a custom Content Management System. When the admin is logged in, they are able to add items, edit items, delete items, upload photos to the photo gallery, and delete photos from the photo gallery.

Minnezoo

This is a mobile app concept design that was completed using OmniGraffle Pro and Photoshop. The idea of this app is to entertain children while they are at the zoo. This project focused on the User Experience design. I interviewed different people at the San Diego Zoo to see how they keep their children entertained on the long days. I then created wireframes and paper prototypes before designing the app. The app allows for the kids to play games, write and draw notes, take pictures, complete quizzes, and look up information about the animals.

TruChart Mobile

This mobile app concept was created using Sketch 3. It is a redesign of an app that I designed while working at Mediture. The app is used by nurses who provide homecare to seniors. It allows them to see their schedule, as well as the tasks that they need to perform.
